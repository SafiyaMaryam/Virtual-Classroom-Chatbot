Java Virtual Classroom Bot Assistant

A Java-based Virtual Classroom Bot Assistant designed to support students and teachers in an online learning environment.
The bot helps automate classroom interactions like answering basic queries, managing schedules, sharing resources, and assisting during virtual sessions.
